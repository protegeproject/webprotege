Welcome to iCAT-PS - the ICPS Collaborative Authoring Tool!

This platform is used for the authoring of the International Classification for Patient Safety alpha draft.

To edit the content of ICPS categories, please go to the ICPS Content tab and select the category in the hierarchy.
To edit the content of ICPS value sets, please go to the Value Sets tab and select the values in the hierarchy.
To view or create notes and discussions related to a category, please go to the Category Notes and Discussions tab.

 Now you can receive notifications about changes and discussions. Click here to find out how.

For a visual tutorial please take a look at our screen cast series explaining the iCAT functionality.
